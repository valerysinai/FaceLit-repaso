# HU-03: Gestión Académica

**Módulo:** RF-3 — Gestión Académica  
**Pantallas:** 6  
**Archivos:** `app/admin/academic/`, `features/academic/`

## Pantallas

| Pantalla | Ruta | Rol |
|----------|------|-----|
| Programas | `/admin/academic` | Admin |
| Registrar Programa | `/admin/academic/programs/register` | Admin |
| Detalle Programa | `/admin/academic/programs/[id]` | Admin |
| Registrar Ficha | `/admin/academic/fichas/register` | Admin |
| Detalle Ficha | `/admin/academic/fichas/[id]` | Admin |
| Unirse a Ficha | `/apprentice/join-ficha` | Aprendiz |

## Reglas

- Programa puede tener múltiples fichas (1:N)
- Ficha pertenece a un único programa
- Aprendiz solo 1 ficha activa
- Código de ficha debe existir y estar activo
