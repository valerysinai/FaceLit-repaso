# HU-01: Inicio de Sesión y Registro

**Módulo:** RF-1 — Accesos al Sistema  
**Pantallas:** 11  
**Archivos:** `app/auth/`, `features/auth/`

## Pantallas

| Pantalla | Ruta | Descripción |
|----------|------|-------------|
| Landing | `/` | Página de inicio |
| Login | `/auth/login` | Email + contraseña + política |
| Registro | `/auth/register` | 13 campos validados |
| Validación Email | `/auth/email-validation` | Código 6 dígitos, 5 min |
| Recuperación | `/auth/password-recovery` | Solicitar código |
| Verificar Identidad | `/auth/verify-identity` | Ingresar código |
| Nueva Contraseña | `/auth/new-password` | Con políticas |
| Confirmación | `/auth/password-reset-done` | Éxito |
| Registro Facial | `/auth/teenager-registration` | Cámara |
| Consentimiento | `/auth/minor-consent` | Para menores |
| Privacidad | `/auth/privacy-notice` | Aviso legal |

## Validaciones

- Nombre/Apellidos: solo letras
- Contraseña: 8-15 caracteres, mayúscula, número, símbolo
- Edad: 8-100 años, consistencia TI/CC
- Email: verificado con código de 6 dígitos (vigencia 5 min)
- Menores: consentimiento obligatorio del acudiente
