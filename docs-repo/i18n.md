# Configuración de Internacionalización (i18n)

## Idiomas Soportados

| Código | Idioma | Archivo |
|--------|--------|---------|
| `es` | Español | `shared/i18n/locales/es.ts` |
| `en` | English | `shared/i18n/locales/en.ts` |
| `de` | Deutsch | `shared/i18n/locales/de.ts` |
| `fr` | Français | `shared/i18n/locales/fr.ts` |

## Cómo Usar

```typescript
import { useTranslation } from 'react-i18next';

function MyScreen() {
  const { t } = useTranslation();
  return <Text>{t('environments.title')}</Text>;
}
```

## Agregar Nuevas Claves

1. Agregar clave en `es.ts` (idioma base)
2. Agregar traducción en `en.ts`, `de.ts`, `fr.ts`
3. Usar en componentes con `t('clave.nueva')`

## Selector de Idioma

El componente `LanguageSelector` en `shared/components/ui/` permite cambiar idioma en tiempo real. La preferencia se persiste en `localStorage`.
